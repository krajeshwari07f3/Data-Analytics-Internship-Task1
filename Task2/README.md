# 📊 Task 2: Exploratory Data Analysis (EDA) on Superstore Dataset

---

## 🎯 Objective
To perform Exploratory Data Analysis (EDA) on the Superstore dataset and extract meaningful insights using Python.

---

## 📖 Description
This task focuses on analyzing the cleaned dataset to understand patterns, trends, and relationships between different variables such as sales, profit, category, and region.

---

## 🛠️ Tools & Technologies
- 🐍 Python  
- 📊 Pandas  
- 📈 Matplotlib  
- 🎨 Seaborn  
- 💻 Jupyter Notebook / VS Code  

---

## 🔍 Steps Performed

1. 📂 Loaded the cleaned dataset  
2. 🔎 Explored data using:
   - `df.head()`
   - `df.info()`
   - `df.describe()`  
3. 📊 Created visualizations:
   - Sales by Category  
   - Profit by Category  
   - Top 10 Products by Sales  
   - Sales by Region  
4. 📈 Analyzed relationships between variables  
5. 💡 Derived meaningful business insights  

---

## 📊 Key Insights

- 📌 Technology category generated the highest sales  
- 📌 Some categories have high sales but lower profit  
- 📌 Top products contribute significantly to total revenue  
- 📌 Sales vary across different regions  
- 📌 Discounts can impact profit negatively

Successfully analyzed the dataset and extracted valuable insights that can help in business decision-making.
