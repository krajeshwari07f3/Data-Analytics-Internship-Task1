# 📊 Task 1: Data Immersion & Wrangling (Data Cleaning)

---

## 📖 Description
This project focuses on cleaning and preparing a dataset for analysis using Python and Pandas.

The dataset used is the Superstore dataset, which contains sales, customer, and product information.

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- Jupyter Notebook / VS Code

---

## 🔍 Steps Performed

1. Loaded the dataset
2. Explored data using:
   - df.head()
   - df.info()
   - df.describe()
3. Checked for missing values
4. Verified duplicate records
5. Cleaned column names
6. Converted date columns into proper datetime format
7. Saved the cleaned dataset

---

## ✅ Key Findings
- No missing values found in dataset
- No duplicate records found
- Dataset is clean and ready for further analysis

- cleaned_superstore.csv → Cleaned dataset
