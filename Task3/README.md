# 📊 Data Analytics Internship - Task 3

## 📌 Project Title
Interactive Dashboard Creation using Power BI

---

## 📖 Description
This project focuses on creating an interactive dashboard using Power BI to visualize key business insights from the Superstore dataset.

The dashboard provides a clear understanding of sales, profit, and quantity across different categories and regions.

---

## 🛠️ Tools Used
- Power BI
- Dataset: Superstore Dataset

---

## 📊 Dashboard Features
- 📈 Sales by Category (Bar Chart)
- 💰 Profit by Category (Bar Chart)
- 🌍 Sales by Region (Bar Chart)
- 🎯 Category Filter (Slicer)
- 📌 KPI Cards:
  - Total Sales
  - Total Profit
  - Total Quantity

---

## 🔍 Insights
- Technology category has highest sales
- Furniture shows lower profit compared to other categories
- West region has highest sales performance

---

- Task3_Dashboard.pbix → Power BI dashboard file
- Screenshot.png → Dashboard preview

## 🚀 Outcome
Developed an interactive and visually appealing dashboard to analyze business performance effectively.
