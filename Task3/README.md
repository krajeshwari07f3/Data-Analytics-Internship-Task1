# 📊 Task 3: Deep-Dive Analysis & Interactive Dashboard

---

## 📖 Description
This project focuses on analyzing sales data and building an interactive dashboard using Excel.

The Superstore dataset was used to understand sales performance across categories, regions, and time.

---

## 🛠️ Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Charts (Bar, Line, Pie)
- Slicers

---

## 🔍 Steps Performed

1. Created pivot tables for analysis  
2. Analyzed sales by category, region, and month  
3. Built different charts:
   - Bar charts for category and region  
   - Line chart for monthly trend  
   - Pie chart for distribution  
4. Added slicers for:
   - Category  
   - Region  
   - Segment  
5. Connected slicers to all charts  
6. Designed and arranged dashboard layout  

---

## 📈 Key Insights
- Technology category has highest sales  
- Sales peak during November and December  
- West region performs the best  
- Sales show an increasing trend over time
